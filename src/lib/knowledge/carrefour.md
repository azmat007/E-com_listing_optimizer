# Carrefour UAE/SA Listing Rules

## Platform
- Mirakl-based ingestion.

## Text Rules
- Short description: max 90 characters.
- Long description: at least 90 characters.
- Bilingual (English + Arabic) fields are explicitly supported.

## Image Rules
- Max file size: 500 kb per image URL.
- Allow up to 6 additional alternate-angle images.

## SEO / Search
- Local UAE/SA terminology.
