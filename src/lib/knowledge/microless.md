# MicroLess Listing Rules

## Platform Focus
- Electronics and hardware-centric marketplace.

## Text Rules
- Listings must feature technically accurate manufacturer specifications.
- Product identifiers (EAN, UPC, Part Numbers) are mandatory.

## Image Rules
- Images must reflect the exact model number variant listed.
- Generic stock family images covering multiple specs are rejected.

## SEO / Search
- Technical spec accuracy is prioritized over marketing fluff.
