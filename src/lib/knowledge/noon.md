# Noon.com Listing Rules

## Text Rules
- Title structure: Brand + Model Name + Product Type + Primary Feature/Color.
- No generic marketing jargon ("High quality", "Guaranteed", "Premium").
- No non-standard symbols in title or bullets.

## Image Rules
- Main image: clean white background, no text overlays, brand badges, or stickers.
- Secondary carousel images: localized lifestyle context images allowed.

## SEO / Search
- UAE and Saudi local terms preferred.
- Keep bullets concise; avoid vague claims.
